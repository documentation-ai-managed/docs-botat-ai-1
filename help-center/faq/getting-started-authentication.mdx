## Overview

You need an API key to use botat ai's API.

## Get Your API Key

1. Log in at `https://dashboard.botatai.com`  
2. Click "API Keys."  
3. Generate a new key and copy it.

## Usage

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" https://api.botatai.com/v1/message
```
