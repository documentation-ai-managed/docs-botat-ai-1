## Quick Start

Get up and running in under 5 minutes.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Create your free account at `https://dashboard.example.com/signup`. No credit card required.
  </Step>
  <Step title="Connect WhatsApp" icon="phone">
    Verify your WhatsApp Business API access. Use your phone number to link instantly.
  </Step>
  <Step title="Configure Bot" icon="settings">
    Customize your AI bot flows. Here's a basic setup example:
    
    <CodeGroup tabs="JSON,JavaScript">
      ```json
      {
        "botName": "SupportBot",
        "welcomeMessage": "Hello! How can I help you today?",
        "intents": ["support", "sales", "info"]
      }
      ```
      ```javascript
      const botConfig = {
        name: 'SupportBot',
        welcome: 'Hello! How can I help?',
        apiEndpoint: 'https://api.example.com/v1/bots'
      };
      ```
    </CodeGroup>
  </Step>
  <Step title="Test & Launch" icon="rocket">
    Send a test message to your number and go live.
  </Step>
</Steps>
